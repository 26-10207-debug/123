# 도화지(DOHWAJI)

발표용 모바일 웹앱 프로토타입입니다. 해외 완제품 대신 공식 라이선스, 설계, 콘텐츠를 디지털로 이동시키고 소비지에 가까운 인증 제작 파트너가 제품을 생산한다는 개념을 시연합니다.

## 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`을 엽니다.

## 빌드 및 점검

```bash
npm run build
npm test
```

## 주요 흐름

- 홈 → Paper Moon 채널 → CEREAL Magazine → 구매하기 → 주문/결제 → 제작 추적
- 탐색 → 검색어 입력 또는 필터 선택 → 결과 필터링
- 구독 → 새 상품 피드 → 상품 상세
- 내정보 → 공식 채널 만들기 → 파일명 표시 및 생성 성공 메시지
